# X-PortGame 2.0 🎮

Una plataforma moderna de juegos online gratuitos, sin anuncios intrusivos y 100% segura.

## Características ✨

- **Sin Anuncios**: Experiencia de juego limpia sin interrupciones publicitarias
- **100% Seguro**: Todos los juegos están verificados y libres de malware
- **Rápido y Ligero**: Carga instantánea sin necesidad de descargas pesadas
- **Responsive**: Funciona perfectamente en dispositivos móviles, tablets y computadoras
- **Interfaz Moderna**: Diseño atractivo y fácil de usar con Tailwind CSS
- **Búsqueda Inteligente**: Encuentra rápidamente tus juegos favoritos
- **Categorías Organizadas**: Navega por diferentes géneros de juegos

## Tecnologías Utilizadas 🛠️

- **HTML5**: Estructura semántica moderna
- **CSS3**: Estilos avanzados con animaciones
- **JavaScript**: Funcionalidad interactiva y dinámica
- **Tailwind CSS**: Framework de utilidades CSS para diseño responsivo
- **Git**: Control de versiones

## Cómo Usar 🚀

### En tu Navegador

1. Abre el archivo `index.html` en tu navegador web
2. Navega por las diferentes secciones
3. Usa la barra de búsqueda para encontrar juegos específicos
4. Haz clic en "Jugar Ahora" para comenzar un juego

### Instalación Local

```bash
# Clonar el repositorio
git clone https://github.com/oliverperezcate34-source/X-PortGame2.0.git

# Navegar al directorio
cd X-PortGame2.0

# Abrir en tu navegador (en Windows)
start index.html

# O en Mac
open index.html

# O en Linux
xdg-open index.html
```

## Personalización 🎨

### Cambiar Colores
Edita el archivo `styles.css` para cambiar la paleta de colores:

```css
/* Cambiar color principal de púrpura a azul */
.from-purple-500 { /* Cambiar a from-blue-500 */ }
```

### Agregar Nuevos Juegos
Edita el array `games` en `script.js`:

```javascript
const games = [
    {
        id: 10,
        title: "Mi Nuevo Juego",
        description: "Descripción del juego",
        category: "accion",
        emoji: "🎮",
        rating: 4.5
    },
    // ... más juegos
];
```
